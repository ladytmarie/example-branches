Basic Repository Example

A repo that has just 5 basic commits on master by three different coders.

  • Fork
  • Add Collaborator
  • Clone git clone
  • Branch git branch [name]
  • Check to see if branch was created git branch
  • Change - Edit file
  • git add *
  • git commit -m "message"
  • git push origin limb (branch name if collaborator)
  • git push origin master (if project manager)
